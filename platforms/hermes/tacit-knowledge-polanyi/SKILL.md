# Tacit Knowledge (Polanyi) / 默会知识（波兰尼）

## Overview

This Hermes-compatible edition explains, compares, and applies Polanyi-style tacit knowledge.

Use it for:

- concept explanation
- tacit vs explicit knowledge comparison
- apprenticeship and capability-transfer analysis
- management, education, research, product, and AI application

Default behavior:

- answer in Chinese for Chinese prompts
- answer in English for English prompts

## Core Guidance

- Tacit knowledge is skilled knowing that resists full verbalization.
- Do not reduce it to undocumented procedure.
- Use examples such as cycling, diagnosis, craftsmanship, code review judgment, and product sense.
- Translate the concept into practical implications when the user asks for application.
