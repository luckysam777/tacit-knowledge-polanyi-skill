# Tacit Knowledge (Polanyi) / 默会知识（波兰尼）

## Overview

This Claude Code edition explains, compares, and applies Polanyi-style tacit knowledge.

Default behavior:

- answer in Chinese when the user asks in Chinese or gives no language preference
- answer in English when the user asks in English
- keep the explanation practical unless the user clearly wants philosophical or academic depth

This skill is built for concept clarification, comparison with explicit knowledge, case analysis, and domain translation into management, education, product work, research, and AI discussion.

## When To Use

Use this skill when the user wants to:

- explain tacit knowledge, explicit knowledge, know-how, or embodied skill
- explain Michael Polanyi or the phrase `we know more than we can tell`
- compare tacit and explicit knowledge
- analyze apprenticeship, craft, intuition, expertise, or expert judgment
- apply tacit knowledge to management, education, product design, research, organizational learning, or AI
- interpret a case, text, or organizational problem through a Polanyi lens

Also trigger when the user asks in Chinese about:

- 波兰尼
- 默会知识
- 缄默知识
- 隐性知识
- 显性知识
- 学徒制
- 技能传承
- 组织学习

## Response Strategy

- define the concept clearly
- compare tacit and explicit knowledge when relevant
- use concrete examples early
- derive practical implications for teaching, management, product work, research, or AI
