# 波兰尼与默会知识速查

## 核心命题

波兰尼最著名的命题可以概括为：人所知道的，往往多于他能够完整说出的。这里的重点不是“知识被故意隐藏”，而是很多高水平行动依赖识别、拿捏、节奏、身体定向、情境判断与熟练感，这些内容无法被完全翻译成明晰规则。

## 术语对照

- **Tacit knowledge**: 默会知识 / 缄默知识 / 隐性知识
- **Explicit knowledge**: 显性知识 / 明示知识 / 可编码知识
- **Know-how**: 会做的能力 / 实践性能力
- **Embodied skill**: 具身技能
- **Apprenticeship**: 学徒制 / 师徒传承

## 一分钟解释版本

默会知识指的是一种“会做但未必能完整说清”的知识。它通常和显性知识并存，而不是取代显性知识。
